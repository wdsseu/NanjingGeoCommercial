# NanjingGeoCommercial

南京市地理演化项目 —— 地块、POI、道路、地铁关联分析可视化


## 项目简介

本项目聚焦南京市城市商业地理演化，通过地块、POI、道路、地铁四个维度的数据，构建地块之间的邻接关系，分析业态分布，并实现道路、地铁与地块的关联展示。前端基于 Leaflet 和 ECharts 进行可视化交互。

## 核心功能

### 1. 地块模块
- 地块以多边形形式展示，可框选。
- 地块之间存在**邻接关系**，可在地图上高亮相邻地块。

### 2. POI 与用地性质
- 每个地块对应多个 POI（商业业态点）。
- 统计地块内各业态占比，取**占比最高的 POI 类型**作为该地块的用地性质（如商业、住宅、办公等）。

### 3. 道路关联
- 每个地块可能邻接多条道路。
- 可视化展示地块与周边道路的关联关系。

### 4. 地铁关联
- 展示地铁站与地铁线。
- 点击地铁站 → 显示其所属地铁线，并高亮半径 **500m 内**的地块。
- 点击地块 → 显示关联的地铁站信息。
